## Descripción
Esta landing utiliza `TailwindCSS` (CDN) y `Alpine.js` para UI interactiva (FAQ desplegable). Incluye mensajes de valor, precios claros, pruebas sociales y CTAs directas hacia demos y contacto.

## Características clave
- Estructura de conversión con CTAs claras y pricing transparente.
- Secciones de valor: problema, demos, cómo funciona, social proof, FAQ.
- Interactividad ligera con `Alpine.js` para FAQ.
- Estilos modernos con `TailwindCSS` desde CDN.
- Tipografía `Inter` por Google Fonts.
- Optimizada para dispositivos móviles con diseño responsive.
